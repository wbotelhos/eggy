jQuery Eggy is a plugin that generates a customizable Easter Egg fired from a character set.
With default options:
$('#eggy').eggy({
  code: ['y', 'e', 'a', 'h', '!']
});

<div id="eggy"></div>
Default options:
code:    []
Character set that fired the animation.
height:  160
Height of the element that will be triggered.
image:   'img/eggy.png'
Background image of the element.
pause:   480
Time that the element is visible on the screen.
side:    'right'
Side of the screen in which the element will be presented. ('left' or 'right')
sound:   'sound/eggy.wav'
Sound to be played when the element appears.
speed:   200
Speed of appearance of the element.
width:   160
Width of the element that will be triggered.
